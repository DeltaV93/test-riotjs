Off Canvas Panel

The contents of any div can be put into a slide out panel.

To initialize:

        
            $('#divid').jqSlideOutPanel();

Settings

        
            $('#divid').jqSlideOutPanel({
              position:"left",
              width:"200px",
              action:"push"            
            });
Option Type Values Default Description
position String "left","right" "right" Determines the side of the screen the panel will be displayed
action String "push", "overlay" "push" Push: Adjust the content of the screen to accomodate the panel.
Overlay: Slides panel on top of content.
width String can use px or % "350px" Initial width of panel when open. Can use either px or %.
showToggle Boolean true | false true Displays tab to toggle panel open and closed.
animateSpeed Number 0-2000 200 The speed that the panel opens and closes. i.e. 1000 = 1 second
showCloseIcon Boolean true | false false Displays a clickable "X" in the top corner of the slide out panel, which closes the panel.
sliderOpenCallback Function null function called after panel is open
sliderClosedCallback Function null function called after panel is closed

Methods

        Example:
        Open overlay panel on the right.
        
          $("#add-note-panel").jqSlideOutPanel("open");
Method Description
open opens the slider
close closes the slider
showToggle shows the toggle icon
hideToggle hides the toggle button
showCloseIcon show the close icon at the top of the slide out panel
hideCloseIcon hide the close icon at the top of the slide out panel

Example:

Click on the arrow tab on the left side of screen.

          
JavaScript:

          
            $("#add-note-btn").on("click",function() {
              $("#add-note-panel").jqSlideOutPanel("open");
            });

            $("#note-cancel-btn").on("click",function() {
              $("#add-note-panel").jqSlideOutPanel("close");
            });

            //initialize left panel
            $('#nav-panel').jqSlideOutPanel({
              position:"left",
              width:"200px",
              action:"push"
            });

            //initialize right panel
            $('#add-note-panel').jqSlideOutPanel({
              position:"right",
              action:"overlay",
              showToggle:false,
              showCloseIcon:true,
              sliderClosedCallback: function () {
                alert("Panel Closed")
              },
              sliderOpenedCallback: function () {
                alert("Panel Opened")
              }
            });
